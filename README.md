# Task 4: SQL for Data Analysis

## Objective

Use SQL queries to extract, manipulate, and analyse structured data using the Superstore dataset.

## Tools Used

- SQLite Online
- SQL
- Superstore Dataset
- Sublime text

## What I Did

- Imported the Superstore dataset into SQLite Online.
- Corrected the dataset import so that the first row was recognised as column headers.
- Used SQL queries to analyse sales and profit performance by region.
- Applied WHERE, HAVING, GROUP BY, and aggregate functions to filter and summarise data.
- Created customer segments using CASE statements and a reusable SQL VIEW.
- Performed an INNER JOIN to combine transaction data with customer segmentation data.
- Used a subquery and DENSE_RANK() window function to identify the top-selling products within each category.
- Created a composite index to demonstrate query optimisation techniques.

## Key Insights

- The West region generated the highest total sales and profit.
- Technology showed the strongest average profit performance in the West region.
- Customers can be grouped into High Value, Medium Value, and Low Value segments based on lifetime spending.
- Product performance varies significantly within categories.
